# jina

`default:jina` proxies the fetch through `https://r.jina.ai/<url>`, requesting markdown via `X-Return-Format: markdown`.

- Keyless by default — works out of the box, rate-limited without a key.
- Attaching a key raises the rate limit; missing credentials never block fetching.
- furl's default provider today, and the first provider builtin tried in furl's canonical `default:*` order, after the [fetch strategies](/plugins/builtins/fetch-strategies).

See [Providers](/providers) for how to set a jina key via `furl providers`. Provider keys aren't yet exposed as `furl plugins` config fields — see [Getting Started](/plugins/getting-started) for how plugin `secret:<name>` config refs work today.
