# furl

- Purpose: Bun + Turborepo monorepo for the `furl` CLI and shared fetch engine.
- Workspaces: `apps/cli`, `packages/core`, `packages/config`.
- Tooling: Bun, Turborepo, Biome, TypeScript, Effect v4 beta.
- Root commands:
  - `bun install`
  - `bun run check`
  - `bun run format`
- Key files:
  - `package.json`
  - `turbo.jsonc`
  - `biome.jsonc`
  - `packages/config/*`
